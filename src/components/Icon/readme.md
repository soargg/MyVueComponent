# Icon

提供一些适量图标库

## 引入

```javascript
import Icon from '../../appassets/components/Icon';
export default {
    components: {
        't-icon': Icon
    },
};
```

## 例子

```html
<t-icon type="arrowDown"/>
```

## API

| 参数   | 说明      | 类型     | 可选值  | 默认值  |
| ---- | ------- | ------ | ---- | ---- |
| type | 字体图标的类型 | string | 参照案例 | ''   |
